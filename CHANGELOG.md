# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2024-01-01

### Added
- Initial project structure with Docker Compose
- Proxy microservice (API Gateway)
- Tokens microservice with JWT authentication
- Notifications microservice with email/SMS support
- Streaming microservice with S3 storage
- Courses microservice with full CRUD and progress tracking
- ChatBot microservice with contextual assistance
- React frontend with MVVM architecture
- Nginx reverse proxy configuration
- PostgreSQL database setup
- Redis caching
- RabbitMQ message queue
- MinIO S3-compatible storage
- Unit tests with 85%+ coverage
- Comprehensive documentation
- Admin panels for all services
- Management commands
- Initial course data fixtures

### Features
- User registration and authentication
